## 💡 Dicas de Otimização

1. **Comprimir imagens:** Use ferramentas como [TinyPNG](https://tinypng.com/) ou [Squoosh](https://squoosh.app/)
2. **Formato WebP:** Para melhor performance, considere usar WebP além de JPG
3. **Dimensões corretas:** Redimensione as imagens para as dimensões especificadas
4. **Qualidade:** Use fotos profissionais quando possível
5. **Consistência:** Mantenha um estilo visual consistente (iluminação, cores)
